# Glossary

**Queen AI**  
Coordination layer responsible for task routing and validation.

**Worker Node**  
Entity executing tasks and earning rewards.

**Task**  
Atomic unit of work in the system.

**Reputation**  
Dynamic trust score reflecting historical performance.

**Model Pool**  
Collection of available models with metadata and performance history.

**Incentive Layer**  
Reward and penalty mechanisms.

**Swarm Intelligence**  
Emergent intelligence from coordinated agents.
