# Task Lifecycle

1. **Task Creation**: User submits task to Queen AI
2. **Task Evaluation**: Queen AI scores task and identifies eligible workers
3. **Task Assignment**: Selected Worker Nodes are assigned the task
4. **Execution**: Workers perform computation / analysis
5. **Result Submission**: Workers send results back to Queen AI
6. **Verification**: Queen AI compares results and resolves discrepancies
7. **Reward Distribution**: Tokens or points are awarded based on performance
8. **Reputation Update**: Workersâ€™ reputations are updated based on accuracy and reliability
